Uyghur Terms
=====================

A list of Uyghur terms in more than 80 different fields such as Physics, chemistry, astronomy .etc.

v0.1   
-Size: 176,106 terms.

File format:

`[Term 1],[field name]`  
`[Term 2],[field name]`  
...  
`[Term n],[field name]`
